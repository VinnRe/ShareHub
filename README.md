# ShareHub

A simple Online Marketplace using Vanilla JS and NodeJS for server

## Features
- Login and Register System
- Sharing items
- Borrowing items
- Update Account Details
- Moderate items, using admin accounts

## Run Locally

Clone the project

```bash
  git clone https://github.com/VinnRe/ShareHub
```

FRONTEND

Go to the project directory

```bash
  cd frontend
```

Install dependencies (You can choose either)

```bash
  npm install
  npm i
```

Run the web application
```bash
    npm run dev
```

BACKEND

Go to backend directory

```bash
  cd backend
```

Install dependencies (You can choose either)

```bash
  npm install
  npm i
```

Request the .env file from the creators, then put it inside the backend folder
```bash
    - backend
        - .env
```

Run the web application
```bash
    npm start
```

That's it!
